### Tech
Este proyecto usa:
* [SocketIO] - Socket.io
* [Alertify] - alertify.js
* [GoogleMaps] - Google Maps Api v3
* [node.js] - evented I/O for the backend
* [Express] - fast node.js network app framework [@tjholowaychuk]
* [Grunt] - GruntJs

### Installation

Install Dependencies:
```sh
$ npm install
```
Start web server:
```sh
$ npm start
```
